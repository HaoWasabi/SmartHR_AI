## DANH SÁCH TÍNH NĂNG CHI TIẾT
**Tính năng chính (MVP)**

*User Features:*

1. Đăng ký / Đăng nhập (Cognito)

2. Upload CV (hỗ trợ PDF, DOCX)

3. Upload Job Description (text hoặc PDF)

4. Chấm điểm CV nhanh (Instant Analysis)

5. Xem báo cáo chi tiết:
    - Điểm tổng thể (0-100)

    - Điểm theo hạng mục: Kinh nghiệm, Kỹ năng, Học vấn, Dự án, Ngôn ngữ...

    - Phân tích từ khóa khớp với JD

    - Điểm mạnh & Điểm yếu
    
    - Gợi ý cải thiện cụ thể

6. Lịch sử phân tích CV

7. So sánh nhiều CV (top 3 CV tốt nhất cho 1 JD)

**Tính năng nâng cao:**

- RAG Search: Hỏi đáp về CV (“Kinh nghiệm React của candidate này như thế nào?”)

- ATS Score (tương thích với Applicant Tracking System)

- Export báo cáo PDF

- Dashboard cá nhân (số CV đã chấm, điểm trung bình theo ngành)s
